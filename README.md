This is a WPF C# application that uses XAML for the user interface.

To run the program, clone this repository into Visual Studio 2022 (recommended) or any compatible IDE/text editor that supports WPF.

Make sure XAMPP is installed so you can use MySQL.

Open phpMyAdmin and create a database named db_medicaremmcm.

Download the provided SQL.

Import the schema into your newly created database.

Build and run the project in Visual Studio.
